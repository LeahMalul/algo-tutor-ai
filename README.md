## Overview

AlgoTutor AI is an AI-powered study assistant that helps students learn faster.

Users upload a PDF document and receive a structured study pack generated by an LLM.

Each study pack contains:

- 📄 Summary
- 🧠 Key Concepts
- ❓ Practice Questions

---

## Features

- Upload PDF study material
- Automatic PDF text extraction
- AI-powered study pack generation
- Modern React frontend
- FastAPI REST API
- Modular architecture
- Type-safe frontend and backend models

## Running the Backend

```bash
cd backend

python -m venv .venv

pip install -r requirements.txt

uvicorn app.main:app --reload
```

---

## Running the Frontend

```bash
cd frontend

npm install

npm run dev
```
